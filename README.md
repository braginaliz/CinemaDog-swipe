# CinemaSwipe

Tinder-подобный интерфейс для выбора фильмов. Свайпайте карточки влево или вправо,
чтобы пропустить или сохранить фильм в список «Буду смотреть».

## Демо
https://braginaliz.github.io/CinemaDog-swipe/

## Функционал

- **Свайп-колода** — стек карточек, верхняя перетаскивается мышью/пальцем
- **Свайп вправо** — сохранить фильм в «Буду смотреть»
- **Свайп влево** — пропустить
- **Кнопки управления** — альтернатива свайпу
- **Клавиатура** — `←` пропустить, `→` сохранить, `Ctrl+Z` отменить
- **Undo** — отмена последнего свайпа
- **Watchlist** — список сохранённых с возможностью удаления
- **Автозагрузка** — новые фильмы подгружаются когда колода заканчивается
- **Сохранение** — watchlist хранится в localStorage
- **Адаптивность** — работает на мобильных (≥375px) и десктопе
- **Моковые данные** — офлайн-режим с подборкой фильмов о собаках (раскомментировать в `api.ts`)

## API

Данные загружаются с [jsonfakery.com](https://jsonfakery.com/movies/random/20).

> Если сервис недоступен (например, без VPN), в файле `src/api/movies.ts`
> раскомментируйте моковую функцию `fetchRandomMovies` с подборкой фильмов о собаках
> и закомментируйте оригинальную.

## Установка и запуск

```bash
# Клонировать репозиторий
git clone 

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

## Возможные улучшения

Проект реализует весь необходимый функционал, но есть направления для дальнейшего развития:

### Производительность и оптимизация
- **Ленивая загрузка изображений** — оптимизация загрузки постеров с Intersection Observer
- **Сжатие изображений** — добавление WebP форматов

### Функциональность
- **История свайпов** — возможность посмотреть все пропущенные фильмы за сессию
- **Категории/жанры** — фильтрация колоды по жанрам
- **Поиск по Watchlist** — быстрый поиск по сохраненным фильмам
- **Детальная страница фильма** — при клике на карточку открывать модальное окно с полной информацией
- **Режим "Случайный фильм"** — если долго не свайпать, предложить случайный

### UI/UX

- **Свайп вверх/вниз** — дополнительные действия (например, "Посмотрю позже" или "Не нравится")
- **Темная тема** — переключение между светлой и темной темой
- **Drag-and-drop в Watchlist** — возможность сортировать сохраненные фильмы вручную
- **Прогресс-бар** — индикатор оставшихся в колоде фильмов

### Тестирование и надежность
- **Юнит-тесты** — покрытие Jest-тестами
- **E2E-тестирование** 
- **Storybook** 

### Инфраструктура
- **CI/CD пайплайн** 
- **Pre-commit хуки** 
- **Аналитика** — отслеживание популярных фильмов (с какими фильмами пользователи взаимодействуют чаще)

### Доступность (a11y)
- **Улучшенная навигация с клавиатуры** — более полная поддержка для пользователей с ограниченными возможностями
- **ARIA-метки** — добавление правильных  описаний для скринридеров
- **Управление фокусом** — правильное поведение фокуса при открытии/закрытии панелей
